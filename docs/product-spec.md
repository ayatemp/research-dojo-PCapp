# Research Dojo Product Spec

## Concept

Research Dojo is an AI research gym. It turns papers, books, notes, and experiment results into understanding questions, strict reviews, revision tasks, research hypotheses, experiment plans, and Codex implementation tasks.

The product rule is strict: AI does not answer first. The user writes first, then the reviewer scores and attacks weak reasoning.

## MVP Scope

1. Project Dashboard
2. Paper Trainer
3. Strict Review
4. Idea to Experiment
5. Codex Task Bridge

## Core Loop

1. Read paper or note
2. Generate questions
3. User answers without seeing the model answer
4. Strict reviewer scores the answer
5. App creates one concrete revision challenge
6. User resubmits
7. Strong ideas become experiments
8. Stable experiments become Codex tasks

## Success Metrics

- Revision count
- First score to final score delta
- Weakness category improvement
- Research hypotheses created
- Codex tasks created
- Ideas reaching actual experiment code
- Paper cards completed
