# Strict Review Rubric

Total: 100 points.

| Category | Points |
| --- | ---: |
| Problem clarity | 20 |
| Novelty vs prior work | 20 |
| Mechanism depth | 20 |
| Limitations and failure cases | 15 |
| Experiment design | 15 |
| Claim sharpness | 10 |

## Decisions

- 90-100: Strong Accept
- 80-89: Accept
- 70-79: Weak Accept
- 60-69: Borderline
- 45-59: Weak Reject
- 30-44: Reject
- 0-29: Strong Reject

## Reviewer Personas

- Theory reviewer: equations, assumptions, objective consistency
- Experiment reviewer: baselines, ablations, metrics, non-cherry-picked setup
- Novelty reviewer: contribution clarity and related work risk
- Implementation reviewer: cost, compute, communication, rare-class failure modes
